# Testing Guide

## Running Tests

### Smoke Tests

```bash
npm run test:smoke
```

### API Tests

```bash
npm run test:api
```

### All Tests

```bash
npm test
```

### Mock API Tests

```bash
npm run test:mock
```

## Test Reports

### Allure Reports

```bash
npm run test:report
```

### Load Testing

```bash
npm run load:all
```
