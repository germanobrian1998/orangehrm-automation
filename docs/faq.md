# FAQ

## How do I add a new test?

1. Create test file in appropriate directory
2. Import Page Objects
3. Write test with Allure decorators
4. Run: `npm test`

## Can I run tests offline?

Yes, use mock API:

```bash
npm run test:mock
```

## How do I view test reports?

```bash
npm run test:report
```

## How do I run load tests?

```bash
npm run load:all
```
