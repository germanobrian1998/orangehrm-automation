# OrangeHRM Test Automation Framework

Complete testing framework for OrangeHRM with E2E, API, and performance tests.

## Features

- **E2E Testing**: Playwright-based end-to-end tests
- **API Testing**: RESTful API testing with axios
- **Performance Testing**: Load testing with k6
- **Mock APIs**: MSW for offline testing
- **Test Reports**: Allure reports with history
- **Load Testing**: Performance baselines and thresholds
- **Documentation**: Complete API and usage documentation

## Quick Start

```bash
# Install dependencies
npm install

# Run tests
npm test

# View reports
npm run test:report
```
